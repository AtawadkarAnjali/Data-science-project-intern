# Trader Behavior vs Market Sentiment

## Objective
Analyze how Bitcoin market sentiment (Fear/Greed) influences trader behavior and performance.

## Dataset
- Historical trader data (Hyperliquid)
- Bitcoin Fear & Greed Index

## Methodology
1. Data cleaning and preprocessing
2. Daily feature engineering (PnL, win rate, trade size)
3. Sentiment alignment
4. Behavioral analysis
5. Trader segmentation
6. Predictive modeling

## Key Insights
- PnL volatility increases during Fear
- Trade frequency increases during Greed
- Winners adapt risk based on sentiment

## Strategy Recommendations
- Sentiment-adaptive leverage control
- Momentum filters during Greed
